A personal link hub and devlog. SvelteKit (Svelte 5) + Tailwind CSS 4, deployed
as a Node server via Docker behind Coolify/Traefik.

Devlog posts are plain markdown read straight off disk — no CMS, no database.
Optional widgets (Spotify now-playing, Discord presence, GitHub activity) each
degrade gracefully when unconfigured.
